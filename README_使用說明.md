# 生產排程反推系統

本系統依「客戶入庫日」往前反推生產排程，會自動扣除：
- 星期六、星期日
- 假日設定工作表中的休假日
- 每日午休時間
- 客戶入庫前預留工作天

## 一、第一次使用
1. 電腦先安裝 Python 3.10 以上版本。
2. 解壓縮本 ZIP。
3. 雙擊「只安裝套件.bat」。
4. 安裝完成後，雙擊「啟動系統.bat」。
5. 瀏覽器會自動開啟系統。

## 二、Excel 必要工作表
### 1. 訂單資料
必要欄位：
- 製令
- 客戶入庫日

建議欄位：
- 客戶
- P/N
- Type
- Category
- 組立地點
- 組立人員
- 組立進度
- 備註
- 發料日
- 入庫日

### 2. 工時設定
必要欄位：
- 工序順序
- 工序名稱
- 標準工時(小時)

可選欄位：
- 負責單位
- 備註

工序順序請由前到後，例如：
1 備料
2 機構組裝
3 管路配管
4 電控配線
5 測試
6 品保檢驗
7 包裝入庫

### 3. 假日設定
必要欄位：
- 日期

可加欄位：
- 假日名稱

## 三、反推邏輯
以客戶入庫日當天下班時間作為完成時間，依工序順序由最後一道工序往前扣除標準工時。
扣除過程不計週六、週日、假日與午休。

## 四、輸出內容
- 排程總表
- 工序明細
- 每日工時負荷
- 工時設定
- 假日設定

## 五、注意事項
- 工時單位一律為「小時」。
- 客戶入庫日請使用真正的 Excel 日期格式。
- 若同一製令不同機型需要不同工時，可複製一份工時設定後再進階改成依 Type 匹配。
- 本版先以單一標準流程反推，適合快速建立基礎排程。
